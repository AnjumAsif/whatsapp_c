  ## Installation
After cloning this repository, migrate to ```whatsapp-flutter-ui``` folder. There, run the following commands:
```bash
  flutter pub get
  open -a simulator (to get iOS Simulator)
  flutter run
  flutter run -d chrome --web-renderer html (to see the best output)
```
